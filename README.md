# Evolved Athlete — Claude Code Setup

Claude Code configuration, plugins, and setup for Evolved Athlete.

## Quick Install

Open Terminal and run:

```bash
curl -fsSL https://raw.githubusercontent.com/evolved-athlete/claude/main/setup.sh | bash
```

Or clone and run locally:

```bash
git clone https://github.com/evolved-athlete/claude.git
bash claude/setup.sh
```

## What Gets Installed

| Component | What It Is |
|-----------|-----------|
| Claude Code | AI-native CLI |
| main-branch | Business workflow engine — research, ads, VSLs, organic content |
| hormozi | Hormozi strategy frameworks — offers, leads, scaling |
| ladder | Personal accountability and goal tracking |
| Notion plugin | Claude ↔ Notion integration |
| Google Drive MCP | Claude ↔ Google Drive integration |

## Daily Workflow

**Business:**

cd ~/Documents/GitHub/kb
claude

**Personal:**

cd ~/Documents/GitHub/brandon-personal

## Skills Reference

| Command | Description |
|---------|-------------|
| `/start` | Main entry — detects state, routes you |
| `/think` | Research, decide, codify into reference files |
| `/ads` | Create Meta ads + compliance review |
| `/vsl` | Write sales video scripts |
| `/organic` | Create Reels, TikTok, carousels |
| `/hormozi` | Apply Hormozi frameworks |
| `/end` | Close session — crystallize insights, commit |
| `/climb` | Daily accountability and goals |
